Política de Privacidade

Gastos BR  ·  Última atualização: junho de 2026

Sua privacidade é nossa prioridade. O Gastos BR foi criado para funcionar offline, sem criar conta — seus dados financeiros ficam armazenados apenas no seu dispositivo.

📱 Dados armazenados localmente

O Gastos BR armazena somente no seu dispositivo:

  • Nome de exibição que você escolheu
  • Transações financeiras (receitas, despesas, transferências)
  • Contas, categorias, metas e orçamentos
  • Foto de perfil (se adicionada)
  • Configurações do app

Esses dados nunca são enviados para nenhum servidor sem sua ação explícita.

☁️ Serviços de terceiros

O app utiliza os seguintes serviços do Google para melhorar a experiência:

  • Firebase Analytics — coleta dados de uso anônimos (telas visitadas, ações realizadas) para entendermos como o app é usado. Nenhum dado financeiro é coletado.
  • Firebase Crashlytics — registra erros e travamentos automaticamente para corrigirmos problemas. Nenhum dado financeiro é incluído nos relatórios.
  • Firebase Remote Config — permite atualizar configurações do app remotamente sem necessidade de atualizar o aplicativo.
  • Google AdMob — exibe anúncios para usuários gratuitos. O AdMob pode coletar dados de publicidade conforme a Política de Privacidade do Google. Usuários Premium não visualizam anúncios.

💳 Compras in-app (Premium)

As assinaturas Premium são processadas pela Google Play Store. O Gastos BR não coleta nem armazena dados de cartão de crédito ou informações de pagamento. Todo o processamento de pagamento é feito pelo Google.

🔒 Segurança

O app suporta bloqueio biométrico (impressão digital / Face ID) para proteger o acesso às suas informações financeiras. Esse recurso utiliza apenas as APIs nativas do Android — nenhuma biometria é armazenada ou transmitida pelo app.

👤 Crianças

O Gastos BR não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados pessoais de crianças.

📤 Exportação e exclusão de dados

Você pode exportar seus dados a qualquer momento em formato CSV (Configurações → Exportar dados). Para excluir todos os dados, utilize a opção "Apagar todos os dados" nas Configurações do app — isso remove permanentemente todas as informações do dispositivo.
