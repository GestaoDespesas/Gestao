Política de Privacidade

Saiba como o GestãoBR coleta, usa e protege suas informações.

Última atualização: Junho de 2026
📱

1. Sobre o GestãoBR

O GestãoBR é um aplicativo de controle financeiro pessoal desenvolvido para usuários brasileiros. Nosso compromisso é oferecer uma ferramenta útil, transparente e que respeite integralmente a privacidade dos nossos usuários.

Esta Política de Privacidade descreve como tratamos as informações geradas durante o uso do aplicativo GestãoBR, disponível na Google Play Store.

📦

2. Dados que coletamos

✅ Seus dados financeiros (transações, metas, orçamentos) são armazenados APENAS no seu dispositivo e nunca enviados para nossos servidores.

O aplicativo pode coletar as seguintes informações:

  • Nome de usuário: Apenas para personalizar a saudação no app. Não é enviado para nenhum servidor.
  • Dados de transações: Despesas, receitas e transferências registradas por você. Armazenados localmente no dispositivo.
  • Dados de diagnóstico: Relatórios de falhas enviados ao Firebase Crashlytics para melhorar a estabilidade do app.
  • Dados de uso anônimos: Métricas agregadas pelo Firebase Analytics para entender como o app é utilizado.
🚫

3. O que NÃO coletamos

Comprometemo-nos a nunca coletar:

  • CPF, RG ou qualquer documento de identificação
  • Dados bancários, número de cartão ou senhas
  • E-mail ou número de telefone
  • Localização geográfica
  • Contatos do dispositivo
  • Fotos além das anexadas voluntariamente a transações
🎯

4. Como usamos as informações

As informações coletadas são usadas exclusivamente para:

  • Garantir o funcionamento correto do aplicativo
  • Identificar e corrigir falhas técnicas (Crashlytics)
  • Melhorar a experiência do usuário com base em dados anônimos de uso
  • Exibir anúncios relevantes para usuários do plano gratuito (via StartApp SDK)
  • Personalizar configurações remotas do app via Firebase Remote Config
📢

5. Publicidade

O GestãoBR exibe anúncios para usuários do plano gratuito por meio do StartApp SDK (Start.io). Ao usar o app gratuitamente, você concorda com a exibição de anúncios.

  • Usuários Premium não visualizam nenhum anúncio.
  • O StartApp pode coletar dados de dispositivo para personalização de anúncios conforme sua própria política de privacidade.
  • Não compartilhamos seus dados financeiros com anunciantes.
🔥

6. Serviços do Firebase (Google)

Utilizamos os seguintes serviços do Firebase:

  • Firebase Crashlytics: Coleta relatórios de falhas anônimos para melhorar a estabilidade.
  • Firebase Analytics: Métricas de uso anônimas e agregadas.
  • Firebase Remote Config: Permite atualizar configurações do app sem nova versão.

Todos esses serviços seguem a Política de Privacidade do Google.

🔐

7. Segurança dos dados

Adotamos as seguintes medidas de segurança:

  • Todos os dados financeiros ficam armazenados localmente no dispositivo do usuário
  • O app suporta autenticação biométrica (impressão digital / Face ID)
  • Nenhuma dado financeiro é transmitido pela internet
  • O tráfego de rede do app usa exclusivamente HTTPS (sem texto claro)
👤

8. Seus direitos

Você tem o direito de:

  • Apagar seus dados: Acesse Configurações → Perfil → Excluir conta para apagar todos os dados do dispositivo.
  • Exportar seus dados: Use a função de exportação CSV (disponível no plano Premium) para obter uma cópia de todas as suas transações.
  • Revogar permissões: Acesse as configurações do Android para revogar permissões concedidas ao app a qualquer momento.
👶

9. Crianças

O GestãoBR não é direcionado a menores de 13 anos. Não coletamos intencionalmente informações de crianças. Se você acredita que seu filho forneceu informações ao app, entre em contato conosco para que possamos tomar as medidas necessárias.

📝

10. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Quando fizermos alterações significativas, notificaremos os usuários dentro do próprio aplicativo. A data de "Última atualização" no topo desta página sempre refletirá a versão mais recente.

O uso contínuo do aplicativo após as alterações constitui aceitação da nova política.
